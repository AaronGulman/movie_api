This API is meant to be a movie database that will give you the description, genre and the directors of the movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of All the movies "/moives" Get None A json object with all movies and their data
Return movies by movie title "/movies/:Title" Get Make sure to use "%20" for spaces in movie titles in URL A json object containing one movie based on a movie title
Return data about movies based off genre "/movies/genres/:genreName" Get None A json object with data about movies that match the searched genre
Return a director by name "/movies/directors/:directorName" Get ake sure to use "%20" for spaces in directors name in URL A json object with data about a director
Create a new user "/users" Post A json object with data about a user to add, in this fromat: { "Username": "Mel", "Password": "password", "Email": "mel@gmail.com", "Birthday": "Month/Day/Year" } A json object with data about the user that was added with an ID: { "Username": "Mel" "Password": "password" "id": [] "FavoriteMovies": [] }
Allow users to update their info based off of their Username "/users/:Username PUT A json object with data about the user that needs to be updated, like: { Username: Username Password: Password Email: Email Birthday: Birthday FavoriteMovies: FavoriteMovies } A json object with all the updated information
Allow users to add movies to their favorites list "/users/:Username/movies/:MovieID" POST json object with movie ID to add to their favorites list A json object with updated data about favorite movies list
Allow users to delete movies from their favorites list "users/:Username/movies/:MovieID DELETE a json object with the movie ID you want removed from favorites list A json object with data about updated movie favorites list
